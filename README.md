# 💼 Employee Income Prediction using Machine Learning

This project predicts whether an employee's income exceeds 50K using Machine Learning.

## 🚀 Features
- Professional Streamlit Dashboard
- Feature Importance Visualization
- Interactive Analytics Dashboard
- LightGBM Machine Learning Model
- Real-time Predictions

## 🧠 Machine Learning Algorithms Used
- Logistic Regression
- Decision Tree
- Random Forest
- AdaBoost
- Gradient Boosting
- XGBoost
- LightGBM
- CatBoost
- KNN
- SVM
- Voting Classifier
- Stacking Classifier
- Bagging Classifier

## 🏆 Final Model
LightGBM Classifier

### Final Test Accuracy:
86.68%
